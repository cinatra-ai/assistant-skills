# Create Scheduled Trigger

## When to use
The user asks to create a trigger, schedule an automation, set up a recurring job, or run something periodically.

## Workflow
1. Understand the trigger's purpose — what should happen and when.
2. Clarify the schedule (once, daily, weekly, quarterly).
3. Identify which tools and data sources the trigger needs.
4. Ask clarifying questions (at most one round) if the outcome is ambiguous.
5. Build and present the trigger configuration before creating it.

## Presentation
Show the trigger plan as:

**Trigger: [Name]**
- **Schedule:** [description]
- **What it does:** [1-2 sentences]
- **Output:** [where results go]

Ask "Shall I go ahead and build it?" before creating.
